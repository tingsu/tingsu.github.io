Tools and Data

  • Themis: Themis is a collection of real-world, reproducible crash bugs (collected from open-source Android apps) and a unified, extensible infrastructure for benchmarking automated GUI testing for Android and beyond.

  • SetDroid: a fully-automated GUI testing tool for detecting system setting-related defects

  • SmartRocket Unit: a commercial unit testing tool for industrial embedded software

  • Stoat: an automated GUI testing tool for Android

  • DroidDefects/CrashAnalysis: dataset of framework-specific exception bugs of Android

  • UI2code: an automated tool for converting GUI image to design code

  • RecDroid: an automated tool for translating bug reports into GUI tests for Android

  • CAUT: a Dynamic Symbolic Execution (DSE)-based test generator for C language
